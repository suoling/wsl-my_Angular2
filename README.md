## 安装依赖

```shell
npm install
```

## 运行

```shell
ng serve --open
```

## 创建组件

```shell
ng generate component component-name
```

## 创建指令|管道|服务|类|接口|模块 等

```shell
ng generate directive|pipe|service|class|guard|interface|enum|module name
```

## 打包编译

```shell
ng build
```

## 测试

```shell
ng test
```

## 端到端测试

```shell
ng e2e
```
